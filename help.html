Welcome to Assemblitz



What is Assemblitz

Assemblitz is a light web-based application that would quickly translate assembly instructions into machine code.


How to use Assemblitz


Translating Assembly Code to Machine Code

  1. Input assembly instructions in the text area provided found at the left side of website.
  2. Click on the "Assemble" button located at the bottom of the page to translate assembly instructions to its machine language equivalent.
  3. The equivalent machine language would be displayed at the upper right part of the page.

Viewing the Stack

  1. Click on the "View Stack" button located at the bottom of the web page to view the stack on a pop-up window.

Using the Assemblitz Instruction Set

  • read
    1. Assembly code that would make the computer read an integer from the keyboard and store it to the variable name provided
    2. Example usage: read number1
  • disp
    1. Allows the user to display the value stored in the variable
    2. Example usage: disp number1
